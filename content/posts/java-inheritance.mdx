## 상속이란

두 개 이상의 클래스를 부모-자식 관계로 만들어 관리하는 것으로, 부모클래스(슈퍼 클래스)와 자식클래스(서브클래스)로 구성된다.

공통된 특징을 부모클래스에 구현하면 자식클래스에서 중복 선언을 방지하고 클래스 계층구조를 효율적으로 관리할 수 있다.

---

## Java 상속의 주요 특징

1. **단일 상속만 지원** - 하나의 자식클래스는 단 하나의 부모클래스만 상속
2. **상속의 횟수 제한 없음** - 계층적 상속 가능
3. **Object 클래스 상속** - 모든 클래스는 자동으로 java.lang.Object 상속

---

## 핵심 개념들

- **super()** : 부모클래스의 특정 생성자를 명시적으로 호출하는 방법
- **오버라이딩** : 부모클래스의 메서드를 자식클래스에서 재정의
- **업캐스팅/다운캐스팅** : 상속 관계의 객체 타입 변환
- **동적바인딩** : 런타임 중에 실행할 메서드를 결정하는 프로세스
