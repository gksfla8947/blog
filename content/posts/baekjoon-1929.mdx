에라토스테네스의 체를 이용하는 문제이다.

```python
M, N = map(int, input().split())

for i in range(M, N+1):
    if i == 1:
        continue
    for j in range(2, int(i**0.5)+1):
        if i % j == 0:
            break
    else:
        print(i)
```

에라토스테네스의 체는 나중에도 많이 나오니 숙지하자
