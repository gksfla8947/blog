## Object 클래스

Object는 모든 클래스들이 상속받고 있는 최상위 계층의 슈퍼클래스이다. 직접 상속을 명시하지 않아도 모든 클래스는 암시적으로 Object를 상속받는다.

### 주요 메서드
- **clone()** : 현 객체와 동일한 객체 반환
- **equals(Object obj)** : 객체 동등성 비교
- **getClass()** : 현 객체의 런타임 클래스 반환
- **hashCode()** : 해시 코드값 반환
- **toString()** : 객체의 문자열 표현 반환

---

## Wrapper 클래스

기본 자료형(int, float, char 등)을 객체 형식으로 다루기 위한 클래스이다. 8가지 종류가 있다: Byte, Short, Integer, Long, Character, Float, Double, Boolean.

### 선언 방식

```java
Integer ten = new Integer(10);
Float pi = new Float(3.14);
```

### 주요 메서드
- **bitCount()** : 이진수 표현에서 1의 개수
- **intValue()** : int 타입으로 변환
- **parseInt(String s)** : 문자열을 정수로 변환
- **toString()** : 매개변수를 문자열로 변환

### 박싱과 언박싱

기본 자료형을 Wrapper 클래스로 변환하는 것을 **박싱**, 역변환을 **언박싱**이라 한다.

```java
Integer ten = new Integer(10);  // 박싱
int numTen = ten.intValue();     // 언박싱
```
