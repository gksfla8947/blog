## PWM이란?

**PWM : Pulse Width Modulation**

디지털 신호를 조절하여 전류량을 아날로그 방식으로 제어하는 것이다.

일정한 Period(주기)에서 디지털 신호의 Duty를 변화시켜 전류량을 제어한다.

---

## 동작 원리

디지털 신호는 0과 1 두 가지 신호밖에 주지 못한다. 따라서 PWM은 매우 빠르게 디지털 신호를 ON/OFF하면서 신호의 세기를 조절한다.

게임을 할 때 버튼을 빠르게 누르면 빠르기나 위력이 높아지는 것과 비슷하게 생각하면 된다.

예를 들어 LED를 PWM으로 제어할 때, Duty가 50%라면 하나의 Period에서 절반은 신호를 ON, 나머지 절반은 OFF시킴으로써 50% 세기의 밝기를 유지한다.

아두이노의 경우 `analogWrite()` 함수를 통해 내장된 PWM 라이브러리를 사용함으로써 **0~255** 사이의 값을 조절할 수 있다.

---

## Fade 예제 코드

```cpp
int ledPin = 9;    // LED connected to digital pin 9

void setup() {
  // nothing happens in setup
}

void loop() {
  // fade in from min to max in increments of 5 points:
  for (int fadeValue = 0 ; fadeValue <= 255; fadeValue += 5) {
    // sets the value (range from 0 to 255):
    analogWrite(ledPin, fadeValue);
    // wait for 30 milliseconds to see the dimming effect
    delay(30);
  }

  // fade out from max to min in increments of 5 points:
  for (int fadeValue = 255 ; fadeValue >= 0; fadeValue -= 5) {
    // sets the value (range from 0 to 255):
    analogWrite(ledPin, fadeValue);
    // wait for 30 milliseconds to see the dimming effect
    delay(30);
  }
}
```
