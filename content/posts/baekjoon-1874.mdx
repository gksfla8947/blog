일단 문제 이해하는 것도 좀 걸렸다..

그리고 코드를 작성했는데 자꾸 시간초과가 뜨길래 고생했는데

정답 변수인 s를 string에서 list형태로 바꾸니 시간초과가 해결됐다.

아무래도 문자열 더하기와 list.append 성능이 상당히 차이가 많이 나는듯...

```python
N = int(input())
stack = []

push_cnt = 1
s = ""
for i in range(N):
    n = int(input())
    while push_cnt <= n:
        stack.append(push_cnt)
        push_cnt += 1
        s += "+\n"
    if n != stack[-1]:
        s = "NO"
        break
    else:
        stack.pop()
        s += "-\n"
print(s)
```

s를 문자열에서

```python
N = int(input())
stack = []

push_cnt = 1
s = []
for i in range(N):
    n = int(input())
    while push_cnt <= n:
        stack.append(push_cnt)
        push_cnt += 1
        s.append("+")
    if n != stack[-1]:
        s = ["NO"]
        break
    else:
        stack.pop()
        s.append("-")
for i in s:
    print(i)
```
