## 클래스와 객체

클래스는 객체의 모양을 선언한 틀이며, 객체는 그 틀 안에 실제로 구현된 실체이다. 사람이라는 클래스와 개별 인물(객체)의 관계로 이해하면 된다.

### 클래스 구성 요소
1. 클래스 선언부
2. 멤버변수(필드)
3. 생성자
4. 메서드

---

## 핵심 개념

- **new** 키워드로 객체 생성
- **메서드 오버로딩**으로 같은 이름의 여러 메서드 정의
- **this** 키워드로 현재 객체 참조
- **this()** 로 다른 생성자 호출
