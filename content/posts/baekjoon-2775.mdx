DP를 활용하는 문제이다.

처음에는 재귀형식으로 풀었다가 시간초과로 퇴짜먹어서 DP방식을 고민했는데 자꾸 어느 부분에서 이상하게 동작하길래 뭔가했는데..

```python
def getRoom(k, n):
    dp = []
    for i in range(k+1):
        if i == 0:
            for j in range(1, n+1):
                dp.append(j)
        else:
            tmp = dp.copy()
            for j in range(n):
                floor_sum = sum([tmp[i] for i in range(0, j+1)])
                dp[j] = floor_sum
    return dp[n-1]


T = int(input())
for i in range(T):
    k, n = map(int, [input() for j in range(2)])
    print(getRoom(k, n))
```

tmp = dp.copy()가 처음에는 tmp = dp였다.

파이썬의 얕은복사와 깊은복사 개념을 적용한 것으로, DP 계산 과정에서 원본 데이터가 변경되는 오류를 해결했다.
