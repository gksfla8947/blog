## 왜 필터가 필요한가?

초음파 센서에서 거리 측정 시 표면의 매끄러움, 주변 방해물, timeout 등 다양한 요인으로 노이즈가 발생한다.

이러한 노이즈를 없애주기 위한 몇 가지 필터들이 있다. 거리에 따른 LED 밝기 조절 예제를 통해 알아보겠다.

---

## 기본 코드

```cpp
#define PIN_TRIG 12
#define PIN_ECHO 13
#define PIN_LED 9
#define INTERVAL 100
#define SOUND_VEL 346.0

#define DIST_MAX 300
#define DIST_MIN 100

float distance;
float timeout;
float scale;
float dist_prev;

unsigned long last_sampling_time;

void setup() {
  pinMode(PIN_TRIG, OUTPUT);
  pinMode(PIN_ECHO, INPUT);
  pinMode(PIN_LED, OUTPUT);
  digitalWrite(PIN_TRIG, LOW);

  Serial.begin(57600);

  distance = 0;
  timeout = (INTERVAL / 2) * 1000.0;
  scale = 0.001 * 0.5 * SOUND_VEL;
  last_sampling_time = 0;
  dist_prev = 0;
}

void loop() {
  if(millis() < last_sampling_time) return;
  distance = USS_Measure(PIN_TRIG, PIN_ECHO);
  Serial.println(distance);
  last_sampling_time += INTERVAL;
}

float USS_Measure(int trig, int echo) {
  float reading;
  digitalWrite(trig, HIGH);
  delayMicroseconds(10);
  digitalWrite(trig, LOW);

  reading = pulseIn(echo, HIGH, timeout) * scale;
  if(reading < DIST_MIN || reading > DIST_MAX) reading = 0;
  return reading;
}
```

이 코드를 실행시켜 Serial Plotter로 확인하면, 범위에 들어왔음에도 불구하고 갑자기 0으로 떨어지는 부분이 생기고, 그래프가 매끄럽지 않은 것을 볼 수 있다.

## 1. 범위기반 필터

```cpp
void loop() {
  if(millis() < last_sampling_time) return;
  distance = USS_Measure(PIN_TRIG, PIN_ECHO);
  if(distance == 0) {
    distance = dist_prev;
  }
  else {
    dist_prev = distance;
  }
  Serial.println(distance);
  last_sampling_time += INTERVAL;
}
```

`dist_prev`라는 직전 측정 값을 저장할 변수를 만들고, distance가 0이 되었을 때 직전 값을 현재 값에 넣어줘 급격하게 0이 되는 것을 방지한다.

필터링 적용 시 0으로 급격히 떨어지는 것은 관찰되지 않으나 여전히 그래프가 매끄럽지 않고 노이즈가 있다.

## 2. 지수가중이동평균 필터 (EMA)

최근 n개의 값을 이용하여 최신의 값은 높은 가중치, 오래된 값은 낮은 가중치로 적용한다.

이를 점화식으로 나타내면 아래와 같다:

```
EMA(1) = DATA(1)
EMA(k) = alpha * DATA(k) + (1 - alpha) * EMA(k-1)
```

코드에서는 아래 한 줄만 추가하면 된다:

```cpp
dist_ema = alpha * distance + (1 - alpha) * dist_ema;
```

`dist_ema`, `alpha` 변수만 추가해주면 된다.

## 3. 중위수 필터

최근 n개의 값을 정렬하여 그 중 중간값을 사용한다.

```cpp
void loop() {
  if(millis() < last_sampling_time + INTERVAL) return;

  dist_raw = USS_measure(PIN_TRIG, PIN_ECHO);
  sample_list[cnt++] = dist_raw;
  for(int i = 0; i < N; i++) {
    for(int j = 0; j < N-1; j++) {
      if(sample_list[j] > sample_list[j+1]) {
        int temp = sample_list[j];
        sample_list[j] = sample_list[j+1];
        sample_list[j+1] = temp;
      }
    }
  }
  if(N % 2 != 0) {
    median = sample_list[(N-1) / 2];
  }
  else {
    median = (sample_list[N / 2 - 1] + sample_list[N / 2]) / 2;
  }
  if(cnt == N) { cnt = 0; }
}
```

n개의 샘플 데이터를 저장할 배열을 선언한 다음, 최근 n개의 값을 넣어 정렬한 뒤 계속 업데이트한다.

n이 짝수일 때는 중간 두 개의 값의 평균을 median으로 설정한다.
