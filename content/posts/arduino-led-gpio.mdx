## GPIO란?

**GPIO(General Purpose Input-Output) : 표준 입출력**

마이크로프로세서(대표적으로 아두이노)의 핀을 입력 또는 출력 용도로 사용할 수 있게 하여 주변장치와 통신할 수 있는 포트이다.

---

## 점멸 제어 코드

0.5초마다 LED를 깜빡거리게 한다 + 실행 후 단 한번 시리얼모니터에 "Hello World" 출력

```cpp
#define PIN_LED 9

unsigned int count, toggle;

void setup() {
  pinMode(PIN_LED, OUTPUT);
  Serial.begin(115200); // Initialize serial port
  while (!Serial) {
    ; // wait for serial port to connect.
  }
  Serial.println("Hello World!");
  count = toggle = 0;
  digitalWrite(PIN_LED, toggle); // turn off LED.
}

void loop() {
  toggle = toggle_state(toggle); //toggle LED value.
  digitalWrite(PIN_LED, toggle); // update LED status.
  delay(500); // wait for 500 milliseconds
}

int toggle_state(int toggle) {
  if(toggle == 0) { toggle = 1; }
  else { toggle = 0; }
  return toggle;
}
```

---

## 주요 함수 설명

- **`pinMode(핀번호, 출력모드)`** — 지정된 핀번호를 어떤 출력모드로 설정할 것인지 결정
- **`digitalWrite(핀번호, LOW or HIGH)`** — GPIO output 기능 수행. HIGH는 5V 출력, LOW는 0V 출력이다.
- **`delay(ms)`** — 설정한 ms만큼 대기
